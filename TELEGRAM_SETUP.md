# Инструкция: Как подключить Mini App к Telegram

Чтобы ваше приложение заработало внутри Telegram, оно должно быть доступно в интернете по защищенному протоколу **HTTPS**.

## ШАГ 1. Получение публичной ссылки

### Способ 1: Временный (для тестов прямо сейчас)
Мы можем использовать сервис "туннелирования", чтобы временно открыть доступ к вашему локальному компьютеру из интернета.
1. Убедитесь, что ваше приложение запущено (`npm run dev`).
2. Откройте **новый** терминал (в VS Code: Terminal -> New Terminal).
3. Выполните команду:
   ```bash
   npx localtunnel --port 5173
   ```
4. Вы получите ссылку вида `https://...loca.lt`. Скопируйте её.
   _(Иногда localtunnel просит ввести пароль IP адреса при первом переходе. Пароль - ваш публичный IP)._

### Способ 2: Постоянный (Vercel / GitHub Pages)
Для реального использования приложение лучше загрузить на хостинг.
Самый простой бесплатный вариант — **Vercel**:
1. Зарегистрируйтесь на [vercel.com](https://vercel.com).
2. Загрузите ваш проект на GitHub.
3. В Vercel нажмите "Add New Project", выберите ваш репозиторий GitHub.
4. Vercel сам соберет проект и выдаст вам красивую ссылку вида `https://my-community-app.vercel.app`.

## ШАГ 2. Настройка в BotFather

1. Откройте бота **[@BotFather](https://t.me/BotFather)** в Telegram.
2. Создайте нового бота (если нет):
   - Отправьте `/newbot`.
   - Придумайте имя и юзернейм (например, `MyCommunityTestBot`).
   - Вы получите **API Token** (сохраните его, пригодится для скриптов, но сейчас не нужен).
3. Создайте кнопку приложения:
   - Отправьте `/newapp`.
   - Выберите вашего бота из меню.
   - Введите название приложения (например, `Community`).
   - Введите описание.
   - Загрузите иконку (по желанию) или нажмите `/empty`.
   - **ВАЖНО**: Когда бот попросит **Web App URL**, вставьте ссылку из Шага 1 (например, вашу ссылку с Vercel или Localtunnel).
   - Придумайте "Short name" (короткое имя) для кнопки, например `start`.

## ШАГ 3. Проверка

После завершения настройки BotFather пришлет вам ссылку вида:
`t.me/ВашБот/start`

Просто перейдите по ней (или перешлите в своё сообщество), и приложение откроется прямо внутри Telegram!
